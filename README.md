# GreenLeaf Veg ‚Äì Online Food Ordering & Table Reservation System

GreenLeaf Veg is a full-stack restaurant web application that allows customers to order food online, reserve tables, and enables the restaurant to manage orders and reservations through an admin dashboard.

This project simulates a real-world restaurant workflow using modern frontend techniques and a robust Spring Boot backend.

# Project Overview

Customers can browse the menu, add items to cart, and place food orders.

Customers can reserve tables by selecting date, time, table number, and number of guests.

The backend stores all orders and reservations in a MySQL database.

An admin dashboard allows viewing all orders and reservations.

The application follows a clean 3-tier architecture.

        ‚Üì Fetch API (AJAX)

        ‚Üì JPA / Hibernate

Frontend handles UI and user interaction

Backend handles business logic and validation

Database stores persistent data

# Validations Implemented

Phone number must be exactly 10 digits

Reservation date:

Cannot be in the past

Can be booked only up to 30 days ahead

Restaurant timing:

Open: 11:00 AM

Close: 11:00 PM

Reservation duration fixed to 1.5 hours

Prevents invalid and incomplete submissions
